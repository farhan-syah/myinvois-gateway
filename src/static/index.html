Welcome to the MyInvois Gateway!

This server simplifies the process of submitting documents to the MyInvois system.

Instead of dealing with complex UBL (XML/JSON) formats, Base64 encoding, and document hashing directly, you can submit your document data using a developer-friendly JSON structure. This gateway handles the necessary translations and formatting for you.

To explore the available API endpoints and see how to submit documents, please visit the API Documentation.
